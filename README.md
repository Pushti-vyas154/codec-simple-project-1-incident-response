## Incident Response Framework

This project follows the **NIST SP 800-61 Incident Response Framework**, which includes:

1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

---

## Playbooks Covered

### Malware Attack

* Detection using SIEM tools
* System isolation and containment
* Malware removal and recovery

### Phishing Attack

* Email analysis and threat identification
* Credential compromise handling
* User awareness and prevention

### Unauthorized Access

* Detection of suspicious logins
* Account compromise handling
* Privilege escalation prevention
